# Лабораторная работа №1: Генетические алгоритмы  
**Тема:** Решение задачи оптимального распределения товаров с использованием генетического алгоритма

## 1. Цель работы

Разработать и исследовать генетический алгоритм для решения задачи распределения товаров от производителей к городам-потребителям с минимизацией транспортных затрат и учётом ограничений на производственные мощности.

Необходимо:
- Необходимо разработать программу на языке python, реализующую генетический алгоритм по предложенному вариантом заданию.
- Провести эксперименты по разным способам скрещивания (не менее 3-х), разным способам мутирования (не менее трех). Результат отобразить в виде графиков
- Моделирование данных производить на основе максимально правдоподобных данных. Т.е. если рассматривается задача, в которой есть калорийность продуктов, то должны использоваться данные о реальных продуктах с реальной калорийностью.
- Предоставить отчет о проделанной работе.

## 2. Постановка задачи

Дано:
- n производителей, каждый с ограниченной производственной мощностью yᵢ
- k городов-потребителей, каждый с объёмом спроса xⱼ
- матрица расстояний d[i][j] между i-м производителем и j-м городом

Требуется:
- назначить каждому городу одного производителя
- минимизировать суммарные транспортные расходы
- не превышать производственные мощности
- желательно минимизировать избыточную мощность (перепроизводство)

Целевая функция (fitness, минимизация):

fitness = транспортные_расходы + штраф_за_избыток_мощности + жёсткие_штрафы_за_нарушения

## 3. Теоретическая часть. Генетические алгоритмы

### Основные понятия

Генетический алгоритм (ГА) — эволюционный метод оптимизации, основанный на принципах естественного отбора и генетики.

Основные компоненты ГА:

| Компонент          | Описание                                                                 | Аналогия в природе          |
|--------------------|--------------------------------------------------------------------------|-----------------------------|
| Особь (individual) | Одно возможное решение задачи (вектор/строка/список)                    | Организм                    |
| Хромосома          | Кодировка решения (в нашей задаче — список индексов производителей)     | Набор генов                 |
| Ген                | Один элемент решения (в нашей задаче — номер производителя для одного города) | Ген                         |
| Популяция          | Множество особей (несколько решений одновременно)                        | Популяция организмов        |
| Приспособленность (fitness) | Числовая оценка качества решения                                   | Выживаемость и размножаемость |
| Отбор (selection)  | Выбор лучших особей для создания потомков                                | Естественный отбор          |
| Скрещивание (crossover) | Обмен частями генов между двумя родителями                          | Скрещивание                 |
| Мутация            | Случайное изменение одного или нескольких генов                         | Мутация                     |
| Элитизм (опционально) | Сохранение лучших особей в следующее поколение без изменений         | —                           |

### Основные этапы работы ГА

1. Инициализация начальной популяции (случайные решения)
2. Оценка приспособленности всех особей
3. Пока не достигнут критерий остановки:
   - Отбор родителей
   - Скрещивание → создание потомков
   - Мутация потомков
   - Оценка новых особей
   - Формирование новой популяции
4. Возврат лучшего найденного решения

### Используемые операторы в работе

| Тип оператора     | Варианты в коде                          | Описание                                                                 |
|-------------------|------------------------------------------|--------------------------------------------------------------------------|
| Скрещивание       | One Point, Two Points, Uniform           | Одноточечное, двухточечное, равномерное скрещивание                     |
| Мутация           | UniformInt с вероятностями 5%, 10%, 20%  | Замена значения гена на случайное новое из диапазона [0..n-1]           |
| Отбор             | Tournament (tournsize=3)                 | Турнирный отбор — выбирается лучший из случайно взятых 3 особей         |
